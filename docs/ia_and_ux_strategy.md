# Secure Fleet Intelligence Agent - UX Strategy

## Visual Direction
Operational Premium: dense, enterprise-grade, high-contrast, mission-control feel.

## Enterprise Side
Desktop-first dashboard with:
- KPI cards
- Alert tables
- Vehicle detail side panel
- Manual library
- AI diagnostics chat
- Privacy review queue
- Audit and exports

## Driver Side
Mobile-first flow with:
- Large buttons
- Route summary
- Assigned vehicle status
- Issue capture
- Manual lookup
- AI assistant
- Offline fallback

## Component Style
- Dark navy and slate surfaces
- Minimal radius
- Compact industrial badges
- Dense tables
- Structured chat cards
- Redaction markers and secure-data chips

## Implementation Notes
Built for React + Tailwind with reusable layouts for enterprise and driver experiences.
